            WWCO ID

            Аккаунт Fandom для связки правок на редактируемых проектах FKAWP.

            WWCO ID (ранее Паспорт WWCO) — учётная запись пользователя на Fandom, позволяющая идентифицировать её владельца, а также связывать правки на проектах FKAWP Editable. Создаётся у пользователей автоматически при регистрации на сервисах "Fandom".

            Партнёрские сайты и приложения поддерживают вход при помощи технологии API.

            WWCO ID

            Аккаунт Fandom для связки правок на редактируемых проектах FKAWP.

            WWCO ID (ранее Паспорт WWCO) — учётная запись пользователя на Fandom, позволяющая идентифицировать её владельца, а также связывать правки на проектах FKAWP Editable. Создаётся у пользователей автоматически при регистрации на сервисах "Fandom".

            Партнёрские сайты и приложения поддерживают вход при помощи технологии API.

            01

            Безопасность

            WWCO ID сообщает обо всех входах в профиль и проверяет пароли на безопасность при каждом вводе. В случае взлома предлагает сменить пароль. Помимо этого, он может посоветовать проверить другие данные: номера телефонов, почту, адреса страниц в соцсетях. Безопасность подтверждается сертификатом соответствия стандарту SOC 2.

            02

            Связка правок

            Использование единого WWCO ID аккаунта, позволяет связывать правки между всеми редактируемыми проектами FKAWP.
